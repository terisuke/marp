## 本の大まかな内容

- 第一章【概要編】
  - marp とは何か？何ができるのか？メリット・デメリットは？を解説します
- 第二章【開発編】
  - 快適にスライドを作るために各種設定をします
- 第三章【作成編】
  - marp でスライドを作るテクニックをお伝えします
- 第四章【発表編】
  - スライド発表における marp の上手な使い方・ツールを紹介します
